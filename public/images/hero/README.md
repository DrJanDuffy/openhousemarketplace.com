# Hero / banner images

Use this folder for hero and banner images on key pages.

| Filename | Suggested use |
|----------|----------------|
| hero-home.jpg | Homepage hero (Summerlin West, open houses, or search CTA) |
| hero-neighborhoods.jpg | Neighborhoods page |
| hero-open-houses.jpg | Open houses / weekend tours |
| hero-new-construction.jpg | New construction page (model homes, builders) |
| hero-luxury.jpg | Luxury homes (e.g. Red Rock, The Ridges) |
| hero-contact.jpg | Contact page (office or “Contact Dr. Jan Duffy”) |

Use **hyper-local** subject matter (Summerlin, Red Rock, Las Vegas) and descriptive **alt text** in the app. Prefer consistent aspect ratio for heroes (e.g. 16:9 or 3:1).

See `public/images/ASSETS.md` for the full checklist.
